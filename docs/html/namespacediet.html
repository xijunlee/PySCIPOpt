diet -- model for the modern diet problem
Parameters:
    - F: set of foods
    - N: set of nutrients
    - a[i]: minimum intake of nutrient i
    - b[i]: maximum intake of nutrient i
    - c[j]: cost of food j
    - d[j][i]: amount of nutrient i in food j
Returns a model, ready to be solved.
