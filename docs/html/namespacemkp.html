mkp -- model for solving the multi-constrained knapsack
Parameters:
    - I: set of dimensions
    - J: set of items
    - v[j]: value of item j
    - a[i,j]: weight of item j on dimension i
    - b[i]: capacity of knapsack on dimension i
Returns a model, ready to be solved.
